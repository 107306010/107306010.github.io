Taiwan Scenic
使用技術或工具：JavaScript、React.js、MaterialUI
我開發了 TaiwanScenic，提供台灣各縣市的景點資訊，此專案利用交通部的 MOTC Transport API V2 獲取景點資訊，並部署在 Heroku 上，提供即時的景點資訊查詢服務。此專案實現了以下功能：
  • 圖片 Lazy loading 與 Infinite scroll：使用 IntersectionObserver API，提升效能與使用者體驗
